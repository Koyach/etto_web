# Image Guide (画像ガイド)

このプロジェクトで使用する画像の配置場所とファイル名のリストです。
指定された場所に指定された名前で画像を保存すると、自動的にサイトに反映されます。

This is a list of where to place images and what to name them.
Please save existing images to the folders below with the specified filenames.

## 1. Hero Section (トップ)
- **Path**: `public/images/hero/background.jpg`
- **Description**: Main background image for the top screen. Ideally dark and textured.

## 2. Origin (産地)
- **Path**: `public/images/origin/noto-landscape.jpg`
- **Description**: Landscape of Noto Peninsula or Sea.

## 3. Concept (哲学)
- **Path**: `public/images/concept/material.jpg`
  - For "Material Soul" (素材の生命力)
- **Path**: `public/images/concept/artisan.jpg`
  - For "Artisan's Silence" (職人の沈黙)
- **Path**: `public/images/concept/tactile.jpg`
  - For "Tactile Memory" (身体的記憶)

## 4. Editions (商品)
- **Path**: `public/images/editions/edition-hero.jpg`
- **Description**: The main product hero shot (Wajima lacquerware ring).

## 9. Privilege (特典)
- **Path**: `public/images/privilege/diamond-rings.jpg`
- **Description**: Image of diamond rings or engagement ring usage.

---

### How to use (使い方)
1. Prepare your images (jpg, png, webp).
2. Rename them to match the filenames above.
3. Drag and drop them into the corresponding folders inside `public/images/`.
4. Refresh the page.

### Troubleshooting
If you see a box with dashed lines saying "Image Not Found", check the path displayed inside that box. That is exactly where the system is looking for the file.
